First off, if you're taking the time to visit this site/personal project of mine, let me say: Thank You!

I really had several goals when designing this project:

  1. I wanted to learn something new.
  2. Learning how to design, build, and deploy a website from scratch was new to me!
  3. Learning how to design, build, and deploy a website from scratch using the Django Web Framework was always on my bucket list. Now if I ever have a cool idea down the line, I don't have to learn how to use Django for the first time, it's in the arsenal.
  4. I wanted to be able to better retain things that I learn by writing about what I learn (because writing about what I learn helps me to remember what I learn).
  5. I wanted to better motivate myself to learn things new (because writing about what I learn encourages me to learn more strangely enough).
  6. I want to be a better, more articulate and 'powerful' writer.
  7. I want other people to have the same opportunity to learn the things that I learn (without having to go through all the hiccups that I went through).

So with these goals in mind, this project seemed to check off a lot of boxes for me.

I'm not really expecting this to be morph into something outlandishly big. But I do think this is going to help me a lot and if this project can help someone else along the way, all the better.

I don't exactly know what the scope of this blog will be. I am definitely going to write some tutorials on web development (because I just went through that all just now). I am definitely going to write about some software engineering principles (because that is I what get paid to do). I am definitely going to write about some other academic interests (probably machine learning and artificial intelligence and natural language processing). I am maybe going to share some stuff about music theory, blues, and maybe even tackle some songs on the piano and guitar. I have no idea where the scope of this blog will end though.

Ultimately, just consider this to be nothing more than just some fun stuff. That's all it really is at the end of the day.

About me personally, my name is Michael Altschuler. I graduated from TCNJ in 2019 with a degree in Computer Science and have been working fulltime as a Software Enginner/Researcher at AT&T since the summer of 2019. That's really all I got to say about myself if I'm being honest. I'm not really good with elevator pitches or autobiographies, I feel too weird doing that. But you can probably piece together the rest based on all the fun stuff I'll share along the way.

Cheers mate!
